Invalid configuration

Data from CRM should contain:

NameDescription
typeType of Power BI element to display. Allowed values are 'tile' or 'report'
dashboardNameName set on dashboard in Power BI containing the element to display. Required if type is 'tile'
tileNameName in Power BI of tile to display. Required if type is 'tile'
dashboardIdId of dashboard in Power BI containing the element to display. Required if type is 'tile'
tileIdId of Power BI tile to display. Required if type is 'tile'
reportNameName in Power BI of report to display. This or id required if type is 'report'
reportIdId of Power BI report to display. This or name required if type is 'report'

Samples

type=tile&dashboardName=myDashboard&tileName=myTile
type=report&reportName=myReport type=tile&dashboardId=myDashboardGuid&tileId=myTileGuid
type=report&reportId=myReportGuid
