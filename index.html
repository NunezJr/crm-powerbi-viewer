What is this

Ever wanted to display Power BI reports and tiles inside Dynamics CRM? Crm-powerbi-viewer aims to do exactly that.

Quickstart

To get started you just need to perform a few simple steps: 1. Download the latest CRM solution file. 1. Import it into your CRM instance. 1. In either a dashboard or a form add a web resource and link it to his_power-bi-viewer.html 1. On data parameters you define what report or dashboard to display. The options are:

Option Description
type 'tile' for showing a Power BI tile and 'report' for a Power BI report.
dashboardId Required for tiles and is the ID of the dashboard holding the tile.
tileId Id of the tile to display. DashboardId is required when using tileId
dashboardName Can be used in place of Id, but you will get in trouble if several dashboards have the same name (the first found will be used). Required if using tileName
tileName Can be used in place of Id, but you will get in trouble if several tiles have the same name (the first found will be used). Requires dashboardName.
reportId Required if type is 'report'.
reportName Required if type is 'report'. Can be used in place of Id, but you will get in trouble if several reports have the same name (the first found will be used).

Sample data parameters: type=tile&dasboardName=SuperCoolBoard&tileName=NicestTileEver type=report&reportName=AllTheDataYouWillEverNeed
